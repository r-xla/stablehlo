# FuncBody

The body of a
[`Func`](https://r-xla.github.io/stablehlo/dev/reference/Func.md),
containing a list of operations.

## Usage

``` r
FuncBody(items = list())
```

## Arguments

- items:

  ([`list()`](https://rdrr.io/r/base/list.html) of
  [`Op`](https://r-xla.github.io/stablehlo/dev/reference/Op.md))  
  The operations in the function body.

## Value

(`FuncBody`)
