# ValueTypes

List of
[`ValueType`](https://r-xla.github.io/stablehlo/dev/reference/ValueType.md)s.

## Usage

``` r
ValueTypes(items = list())
```

## Arguments

- items:

  ([`list()`](https://rdrr.io/r/base/list.html) of
  [`ValueType`](https://r-xla.github.io/stablehlo/dev/reference/ValueType.md))  
  The types of the values.

## Value

`ValueTypes`
