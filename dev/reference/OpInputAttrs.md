# OpInputAttrs

List of
[`OpInputAttr`](https://r-xla.github.io/stablehlo/dev/reference/OpInputAttr.md)s.

## Usage

``` r
OpInputAttrs(items = list())
```

## Arguments

- items:

  ([`list()`](https://rdrr.io/r/base/list.html) of
  [`OpInputAttr`](https://r-xla.github.io/stablehlo/dev/reference/OpInputAttr.md))  
  The attributes that can be used as inputs to operations.

## Value

(`OpInputAttrs`)
