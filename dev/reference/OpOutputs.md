# OpOutputs

List of
[`OpOutput`](https://r-xla.github.io/stablehlo/dev/reference/OpOutput.md)s.

## Usage

``` r
OpOutputs(items = list())
```

## Arguments

- items:

  ([`list()`](https://rdrr.io/r/base/list.html) of
  [`OpOutput`](https://r-xla.github.io/stablehlo/dev/reference/OpOutput.md))  
  The outputs of an operation.

## Value

(`OpOutputs`)
