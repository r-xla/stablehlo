# OpInputValues

List of
[`OpInputValue`](https://r-xla.github.io/stablehlo/dev/reference/OpInputValue.md)s.

## Usage

``` r
OpInputValues(items = list())
```

## Arguments

- items:

  ([`list()`](https://rdrr.io/r/base/list.html) of
  [`OpInputValue`](https://r-xla.github.io/stablehlo/dev/reference/OpInputValue.md))  
  The values that can be used as inputs to operations.

## Value

(`OpInputValues`)
