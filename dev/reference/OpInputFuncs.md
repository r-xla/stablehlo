# OpInputFuncs

List of
[`OpInputFunc`](https://r-xla.github.io/stablehlo/dev/reference/OpInputFunc.md)s.

## Usage

``` r
OpInputFuncs(items = list())
```

## Arguments

- items:

  ([`list()`](https://rdrr.io/r/base/list.html) of
  [`OpInputFunc`](https://r-xla.github.io/stablehlo/dev/reference/OpInputFunc.md))  
  The functions that can be used as inputs to operations.

## Value

(`OpInputFuncs`)
