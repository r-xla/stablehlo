# OpSignature

This represents the signature of an operation, defining its input and
output types.

## Usage

``` r
OpSignature(input_types, output_types)
```

## Arguments

- input_types:

  ([`ValueTypes`](https://r-xla.github.io/stablehlo/dev/reference/ValueTypes.md))  
  The types of the inputs.

- output_types:

  ([`ValueTypes`](https://r-xla.github.io/stablehlo/dev/reference/ValueTypes.md))  
  The types of the outputs.

## Value

(`OpSignature`)
