# Digamma Operator (CHLO)

This op is from the CHLO dialect, a higher-level companion to stableHLO
that is lowered to stableHLO during compilation. See
<https://openxla.org/stablehlo/generated/chlo#chlodigamma_chlodigammaop>
for details.

## Usage

``` r
infer_types_digamma(operand)

hlo_digamma(operand)
```
