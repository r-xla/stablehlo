# Op

This represents a StableHLO operation.

## Usage

``` r
Op(
  name = OpName(),
  inputs = OpInputs(),
  outputs = OpOutputs(),
  signature = OpSignature()
)
```

## Arguments

- name:

  ([`OpName`](OpName.md))  
  The name of the operation.

- inputs:

  ([`OpInputs`](OpInputs.md))  
  The inputs to the operation.

- outputs:

  ([`OpOutputs`](OpOutputs.md))  
  The outputs of the operation.

- signature:

  ([`OpSignature`](OpSignature.md))  
  The signature of the operation.

## Value

(`Op`)
