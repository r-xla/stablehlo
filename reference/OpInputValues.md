# OpInputValues

List of [`OpInputValue`](OpInputValue.md)s.

## Usage

``` r
OpInputValues(items = list())
```

## Arguments

- items:

  ([`list()`](https://rdrr.io/r/base/list.html) of
  [`OpInputValue`](OpInputValue.md))  
  The values that can be used as inputs to operations.

## Value

(`OpInputValues`)
