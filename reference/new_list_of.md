# Create a new list_of class

Create a new list_of class

## Usage

``` r
new_list_of(class_name, item_class, validator = NULL)
```

## Arguments

- class_name:

  Name of the new class

- item_class:

  Name of the class items should be (for documentation)

- validator:

  Optional function to validate items

## Value

Constructor function for the list_of subclass
