# OpInputAttr

This represents an attribute that can be used as input to an operation.

## Usage

``` r
OpInputAttr(name = character(0), value = Constant())
```

## Arguments

- name:

  (`character(1)`)  
  The name of the attribute.

- value:

  (`Constant`)  
  The value of the attribute.

## Value

(`OpInputAttr`)
