# Convert R value to Constant

Convert R value to Constant.

## Usage

``` r
r_to_constant(value, dtype = NULL, shape, ...)
```

## Arguments

- value:

  (any)  
  The value to convert.

- dtype:

  (`character(1)`)  
  The dtype of the constant.

- shape:

  ([`integer()`](https://rdrr.io/r/base/integer.html))  
  The shape of the constant.

- ...:

  (any) Additional arguments.
