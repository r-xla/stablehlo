# Convert to TensorDataType

Convert to TensorDataType.

## Usage

``` r
as_dtype(x)
```

## Arguments

- x:

  (any)  
  Object to convert. Can currently be a string (one of pred, i8, i16,
  i32, i64, ui8, ui16, ui32, ui64, f32, f64) or a
  [`TensorDataType`](TensorDataType.md) object.

## Value

`TensorDataType`
