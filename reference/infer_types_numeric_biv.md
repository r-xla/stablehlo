# Infer types for numeric binary operations

Infer the types for numeric binary operations.

## Usage

``` r
infer_types_numeric_biv(lhs, rhs)
```

## Arguments

- lhs:

  (`ValueType`)  
  The left-hand side operand.

- rhs:

  (`ValueType`)  
  The right-hand side operand.

## Value

(`ValueType`)  
The inferred type.
