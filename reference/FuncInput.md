# FuncInput

This represents an input of a [`Func`](Func.md).

## Usage

``` r
FuncInput(id = ValueId(), type = ValueType(), alias = NULL)
```

## Arguments

- id:

  ([`ValueId`](ValueId.md))  
  The id of the input.

- type:

  ([`ValueType`](ValueType.md))  
  The type of the input.

- alias:

  (`integer(1)` \| `NULL`)  
  With which output buffer to alias this input.

## Value

(`FuncInput`)
