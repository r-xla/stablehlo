# Get the current module

Get the current module created via
[`hlo_module`](https://r-xla.github.io/stablehlo/reference/hlo_module.md)
or
[`local_module`](https://r-xla.github.io/stablehlo/reference/hlo_module.md).

## Usage

``` r
.current_module()
```

## Value

A [`Module`](https://r-xla.github.io/stablehlo/reference/Module.md)
object.
