# Infer types for boolean binary operations

Infer the types for boolean binary operations.

## Usage

``` r
infer_types_integerish_biv(lhs, rhs)
```

## Arguments

- lhs:

  (`ValueType`)  
  The left-hand side operand.

- rhs:

  (`ValueType`)  
  The right-hand side operand.

## Value

(`ValueType`)  
The inferred type.
